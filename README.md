# 7_mistery_fix

Функция get_roots() находит решение квадратного уравнения.

#Использование.
Импортируем функцию:
`from quadratic_equation import get_roots`
Находим корни:
`root1, root2 = get_roots(a, b, c)`, где **a**, **b**, **c** постоянные (числовые) коэффициенты уравнения вида **ax^2 + bx + c = 0**.
